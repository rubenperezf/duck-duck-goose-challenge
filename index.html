Duck-Duck-Goose Challenge

Instrucction:

  • Using JavaScript, create an array of 100 numbers, starting with 1 ending with 100.
  • Replace every number that is divisible by 3 with Duck
  • Replace every number this is divisible by 5 with Goose
  • Replace every number that is both divisible by 3 and 5 with DuckGoose
  • HINT: using the ‘modulus’ operator you can determine what a number is divisible

Click the button to trigger a function that will output array results to a p element with id="demo". It will also output using alert, and provided write function.
